# [1.0.0] - 2026-06-30

## Added

### Analysis Engine

- PHP log parser.
- Line-by-line processing to support large log files.
- Error normalization.
- Smart error prioritization.
- Filters by level, file, and text.
- Analysis engine (`AnalyzerEngine`).
- Unified result model (`AnalysisResult`).

### Exporters

- Console output.
- JSON export.
- CSV export.
- Interactive HTML dashboard.

### Dashboard

- Key performance indicators (KPIs).
- Normalized issues chart.
- Searchable interactive table.
- Dark theme.

### CLI

- `--serve`
